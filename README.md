# Decision Tree to DMN

This project:
- Trains a Decision Tree on the Iris dataset
- Extracts IF-THEN rules
- Converts them into DMN XML
- Generates a visualization using Graphviz

## How to run
python iristree.ipynb

## Output
- dmn_decision_table.dmn
- decisiontree_graphviz.png

## Libraries used
- scikit-learn
- pandas
- numpy
- graphviz
- matplotlib
- Pillow
